# .github

Org profile for [Reloaded-Project](https://github.com/Reloaded-Project).

`profile/README.md` is what shows up on the org homepage. This file is
just for people who wander into this repo directly.

## Updating

```bash
git add profile/README.md
git commit -m "Update org profile"
git push
```

Repo must be public for the profile to render. Stars in the profile README
are a snapshot, update when you feel like it.
